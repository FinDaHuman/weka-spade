# ============================================================
# TEST CASE 08 - SPMF Format with Itemsets
# Format  : SPMF standard sequential format (plain text)
# Encoding: A=1, B=2, C=3, D=4, E=5
# Multiple items before a -1 = they belong to same event (itemset)
# ============================================================
# Seq 1: {A,B} -> {C} -> {D,E}
1 2 -1 3 -1 4 5 -1 -2
# Seq 2: {A} -> {C,D} -> {E}
1 -1 3 4 -1 5 -1 -2
# Seq 3: {A,B} -> {D,E}
1 2 -1 4 5 -1 -2
# Seq 4: {B} -> {C} -> {E}
2 -1 3 -1 5 -1 -2
# Seq 5: {A,B} -> {C,D} -> {E}
1 2 -1 3 4 -1 5 -1 -2
